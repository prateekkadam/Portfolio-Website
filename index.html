My Projects

Movie Recommendation System

As a student, I had the opportunity to work on a movie recommendation system project as part of my studies. The project involved understanding and implementing algorithms that analyzed user preferences and movie data to generate personalized movie recommendations. By collaborating with a team of classmates, we designed and developed a user-friendly web application that allowed users to rate movies, view recommended movies based on their preferences, and explore various genres. The goal of the project was to enhance the movie-watching experience by providing tailored recommendations to users based on their interests and viewing history.

Bike Rental Management System

During my time as a student, I participated in a software development project focused on building a bike rental management system. The project aimed to simulate a real-world scenario for a bike rental company, where users could rent bikes, manage bookings, and handle inventory. Working in a team, we designed and implemented a web application that allowed users to browse available bikes, select a rental period, make reservations, and process payments securely. The project helped us gain hands-on experience in developing practical solutions, understanding user requirements, and building a functional system for managing bike rentals.
